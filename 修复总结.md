#### 智能中文检测
```python
# 自动检测中文字符数量，如果太少则返回标准中文回答
chinese_chars = re.findall(r'[\u4e00-\u9fff]', text)
if len(chinese_chars) < 3:
    if "你是谁" in text:
        return "我是中文AI助手，很高兴为您服务！"
    else:
        return "抱歉，我必须用中文回答您的问题。"
```
